Pict Example
OAuth Login
pict-section-login
This example uses pict-section-login with OAuth enabled (ShowOAuthProviders: true). It requires:
  • A running Orator server with orator-authentication
  • OAuth providers configured (e.g. Google, Microsoft) in the server’s OAuthProviders option
On load the section queries /1.0/OAuth/Providers and renders branded buttons for each configured provider alongside the standard username/password form.
